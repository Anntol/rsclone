## Локальный запуск
1. Создайте файл `.env` в корне приложения
2. В созданном файле укажите переменные окружения:
```
MONGO_CONNECTION_STRING=<адрес вашей локальной или облачной mongodb>
SECRET_TOKEN=<ваш секретный ключ для подписи JWT>
```
3. npm install
4. npm run devbuild
